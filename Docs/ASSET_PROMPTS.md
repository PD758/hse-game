# Asset Generation Prompt

Use this base prompt for a coherent first tileset:

Pixel art top-down 2D roguelike dungeon tileset, 16x16 tiles, dark readable floor, stone walls, locked wooden door, iron gate open and closed, brass key, pressure plate, pushable stone block, spike trap, cyan exit stairs, player adventurer, three enemy states patrol/investigate/hunt, restrained tactical palette, high contrast silhouettes, no text, no UI, transparent background, orthographic, game-ready sprite sheet.

Needed sprites:

- floor variations;
- wall and corner walls;
- closed gate and open gate;
- locked door;
- key;
- pressure plate idle and pressed;
- pushable stone;
- spike trap idle and sprung;
- exit stairs;
- player idle;
- enemy patrol, investigate, hunt.
